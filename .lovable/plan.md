

# خطة: ردود تلقائية لأزرار القوالب + تحديث حالة الشحنة

## الفكرة

لما العميل يدوس على زر Quick Reply في قالب الواتساب (زي "جاهز للاستلام" أو "الغاء الطلب")، النظام هيعمل 3 حاجات تلقائياً:
1. يرد على العميل برسالة محددة مسبقاً
2. يغير حالة الشحنة في جدول المتابعة (مثلاً من "بانتظار المتابعة" لـ "تم الحل")
3. الـ AI لما يشتغل على قسم المتابعة يكون فاهم إن العميل دوس على زر معين ويكمل المحادثة على الأساس ده

---

## التغييرات المطلوبة

### 1. جدول جديد: `followup_button_actions`
يخزن إعدادات كل زر Quick Reply وإيه الى بيحصل لما العميل يدوس عليه:

| Column | Type | Description |
|--------|------|-------------|
| id | uuid | PK |
| tenant_id | uuid | البراند |
| template_id | uuid | FK → followup_wa_templates |
| button_title | text | نص الزر زي ما هو في ميتا (مثلاً "جاهز للاستلام") |
| auto_reply_text | text | الرد التلقائي للعميل |
| update_status_to | text | الحالة الجديدة في جدول المتابعة (pending/contacted/resolved/escalated/cancelled) |
| is_active | boolean | مفعل ولا لا |

### 2. تحديث الـ Webhook (`whatsapp-webhook`)
- لما الويبهوك يستقبل رسالة نوعها `interactive.button_reply` أو `button`:
  - يدور على الـ `button_title` في جدول `followup_button_actions`
  - لو لقاه: يرد تلقائياً بالرسالة المحددة + يحدث حالة الشحنة في `shipment_tracking` بناءً على رقم تليفون العميل
  - الأولوية: أزرار المتابعة قبل قواعد الكلمات المفتاحية العادية

### 3. واجهة إعدادات الأزرار (`FollowupWATemplates.tsx`)
- في كل قالب، تحت قسم الأزرار، يظهر جزء جديد "ردود الأزرار التلقائية"
- لكل زر Quick Reply: حقل لنص الرد التلقائي + اختيار الحالة الجديدة في المتابعة
- مثال:
  - زر "جاهز للاستلام" → رد: "تمام هنتواصل مع شركة الشحن..." → حالة: `resolved`
  - زر "عندي استفسار" → رد: "اتفضل، إيه استفسارك؟" → حالة: `contacted`
  - زر "الغاء الطلب" → رد: "تم تسجيل طلب الإلغاء..." → حالة: `cancelled`

### 4. تحديث الـ AI (`ai-reply`)
- لما الـ AI يشتغل على قسم المتابعة، هيكون في الـ system prompt معلومات عن:
  - إن العميل ممكن يكون رد على قالب واتساب بزر معين
  - الرسالة الى هي نص الزر (مثلاً "جاهز للاستلام") معناها إيه
  - لو العميل دوس "عندي استفسار" الـ AI يكمل المحادثة ويساعده
